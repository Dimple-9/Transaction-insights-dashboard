# Smart Transaction Search & Spending Insights Dashboard

## Overview
This project transforms raw bank transaction data into meaningful financial insights using automated categorization, smart search functionality, and visual analytics. It helps users understand spending behavior and make informed financial decisions.

---
## Problem Statement
Bank transaction statements are difficult to interpret. Users struggle to identify spending patterns, track expenses, and understand where their money goes.

---

## Solution
This project processes transaction data and provides:

automated expense categorization  
smart keyword search  
monthly spending analysis  
category-wise spending insights  
visual dashboard for financial awareness  

---

## Key Features

### Intelligent Categorization
Transactions are automatically classified using merchant keyword recognition.

### Smart Transaction Search
Users can quickly find transactions using keywords such as merchant names or categories.

### Spending Analysis
Monthly trends and category breakdowns help identify spending behavior.

###  Visual Insights Dashboard
Charts and summaries present financial insights clearly.
